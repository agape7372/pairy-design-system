# Pairy Design System

Pairy 프로젝트의 버튼 인터랙션 및 애니메이션 데모 모음입니다.

## 개발 환경 설정

```bash
npm install
npm run dev
```

## 페이지 구성

- `/` - 홈
- `/buttons` - 30가지 좋아요/북마크 버튼 변형
- `/physics` - 물리 기반 버튼 애니메이션
- `/animations` - 모션 및 트랜지션 데모

## 배포

GitHub Pages로 정적 배포됩니다:

```bash
npm run build
```

## 기술 스택

- Next.js 15
- React 19
- Framer Motion
- Tailwind CSS v4
- Lucide Icons
